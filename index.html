Common Scam Patterns

  • Fake shipping labels with incorrect return addresses
  • Overpayment scams with fake checks
  • Phishing attempts through fake return portals
  • Returning different/cheaper items than purchased

Protection Tips

  • Always use your own return shipping labels
  • Verify returned items match the original purchase
  • Never provide personal information via email
  • Use secure payment methods with buyer protection
